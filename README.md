## What It Does

Polygon Hack is a Windows-oriented script utility built for Polygon. It brings together combat aids, visibility overlays, movement modifiers, and inventory automation in one script-based package. The idea is to offer a configurable automation layer and in-game support that can be adapted to different play styles.

The current release focuses on adjustable aim support, ESP-style visibility overlays, recoil and spread handling, and movement tools such as speed changes and teleport nodes. It also includes practical actions like auto heal and inventory management, with updates intended to keep the utility aligned with Polygon 2025 and later Windows use cases.
